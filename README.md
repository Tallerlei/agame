# Epic Adventure ⚔️

A browser-based RPG game featuring fighting and exploration with unique characters, special abilities, and an item system.

## Features

### 🎮 Character System
- **4 Unique Classes**: Warrior, Mage, Rogue, and Healer
- **Character Development**: Level up through combat and quests
- **Stats System**: Strength, Agility, Intelligence, Attack Power, Defense
- **Special Abilities**: Each class has unique abilities that unlock and improve

### ⚔️ Combat System
- **Turn-based Combat**: Strategic fighting with attack choices
- **Ability Usage**: Use class-specific abilities with cooldowns
- **Enemy Variety**: Fight goblins, orcs, skeletons, wolves, and more
- **Victory Rewards**: Earn experience, gold, and item drops

### 📜 Quest System
- **Multiple Quests**: Various quests with different objectives
- **Difficulty Levels**: Easy, Medium, Hard, and Legendary
- **Objectives**: Defeat enemies, explore locations, collect items
- **Rewards**: Experience, gold, and special items

### 🎒 Item System
- **Weapons**: Swords, axes, daggers, staffs, and more
- **Armor**: Head, chest, legs, and feet equipment
- **Consumables**: Health potions and other useful items
- **Trinkets**: Accessories with stat bonuses
- **Rarity Tiers**: Common, Uncommon, Rare, Epic, Legendary

## Play Online

🎮 **[Play Epic Adventure](https://tallerlei.github.io/agame/)** - The game is deployed to GitHub Pages and updated automatically whenever changes are pushed to the master branch.

## Getting Started

### Prerequisites
- Node.js 18 or higher
- npm

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm start
```

Navigate to `http://localhost:4200/` in your browser.

### Building

```bash
npm run build
```

### Testing

```bash
npm test
```

## Technology Stack

- **Framework**: Angular 18
- **Language**: TypeScript
- **Styling**: CSS with custom game-themed design
- **CI/CD**: GitHub Actions for automatic deployment to GitHub Pages

## Game Design

The game follows a classic RPG formula with modern web technologies:

1. **Create your hero** - Choose a name and class
2. **Fight enemies** - Engage in turn-based combat
3. **Complete quests** - Explore the world and complete objectives
4. **Collect items** - Find and equip weapons, armor, and trinkets
5. **Level up** - Grow stronger through experience
